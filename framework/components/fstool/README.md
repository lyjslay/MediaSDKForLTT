# CVI_FSTOOL

CVI_FSTOOL is a simple tool that can recover the broken filesystem(fat)

## Prerequisites


## How to build

```bash
$ make
$ make install
# cvi_fstool shared library and header file will be install in the install directory
```

## How to test

1. build
```bash
$ make
```

2. execute
   
```bash
$ ./sample_fstool
```
